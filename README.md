# Responsive Gallery Template (Free Version)

A clean, beginner-friendly responsive image gallery built with HTML, CSS, and JavaScript.

Stop struggling with image galleries — this template gives you a simple, modern layout with a built-in lightbox that’s easy to customize and use in your own projects.

Built with clean, beginner-friendly code — perfect for learning or quickly adding a gallery to your site.

---

## ✨ Features

- Responsive image grid
- Click-to-open lightbox modal
- Close with ESC key
- Close by clicking outside the image
- Clean and simple design
- Beginner-friendly code

## 📦 How to Use

1. Download or clone this repository
2. Open `index.html` in your browser
3. Replace images in the `/images` folder
4. Update image paths in `script.js`
5. Customize styles in `styles.css`
