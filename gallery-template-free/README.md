Responsive Gallery Template

Thanks for downloading!

This product includes:

FREE VERSION:
- Responsive gallery grid
- Lightbox modal
- Close with ESC and click outside

PRO VERSION:
- Previous / Next navigation
- Keyboard arrow controls
- Image captions
- Image counter
- Enhanced UI and styling

How to use:
1. Open the FREE or PRO folder
2. Replace images in the /images folder
3. Update image paths in script.js
4. Customize styles in styles.css

If you found this useful, consider upgrading to the Pro version for more features.
